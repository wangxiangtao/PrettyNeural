PrettyNeural

A Deep Learning Library written in Scala.

PrettyNeural is a deep learning library written in Scala. It is a sub-library of PrettyMining. PrettyNeural include Multilayer Perceptron, Restricted Boltzmann Machine and Deep Belief Network. The algorithms can be apply on big data set by multithreading or distribution on Spark framework. It also can be a plugin for Weka.

Please note this library is still under construction


Features

  • Neural Network Types:
    Multilayer Perceptron, Restricted Boltzmann Machine and Deep Belief Network
  • Training algorithms: Back Propagation , Contrastive Divergence
  • Activation functions: Linear, Sigmoid(Logistic)
  • Using an Approximate Sigmoid for speed improvement
  • Enable Multithreading though Google Guava library .
  • Supporting running on the top of Weka
  • Distribution on Spark Framework (under construction)
