PrettyMatch is a deep learning library written in Scala. It is a sub-library of PrettyMining. PrettyNeural include Multilayer Perceptron, Restricted Boltzmann Machine and Deep Belief Network. The algorithms can be appply on big data set by multithreading or distribution on Spark. It also can be a plugin for Weka.

Features

  • Neural Network: Multilayer Perceptron, Restricted Boltzmann Machine and Deep Belief Network
  • Training algorithms: Back Propagation , Contrastive Divergence
  • Activation functions: Linear, Sigmoid(Logistic) and Approximate Sigmoid
  • Enable Multithreading though Google Guava library .
  • Supporting run on the top of Weka by plugin
  • Distribution on Spark Framework (under construction)
